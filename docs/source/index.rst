Causal Lasso documentation
========================================

This repository implements paper "A Bregman Method for Structure Learning on Sparse Directed Acyclic Graphs" by Manon Romain and Alexandre d'Aspremont. Paper is available https://arxiv.org/abs/2011.02764

Requirements
-------------
This package requires ``numpy``, ``scipy``, ``tqdm``, ``networkx`` and for now ``mosek``, installation can be done with 
``pip install -r docs/requirements.txt``
